## Overview
Use this checklist before every release to prevent avoidable regressions.

## Checklist
- Validate frontmatter for all content sections.
- Run lint and unit tests.
- Verify dynamic pages have static params where needed.
- Check metadata, sitemap, and robots outputs.
- Test mobile navigation and search behavior.
