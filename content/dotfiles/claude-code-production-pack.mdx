## Overview
This pack is for high-signal coding sessions where you want predictable outcomes: clear plan, bounded execution, and explicit verification.

## Prompt Set

<PromptBlock agent="planner">
Write an implementation plan in phases with explicit dependencies, risk flags, and validation steps.
</PromptBlock>

<PromptBlock agent="executor">
Implement phase by phase. After each phase run lint, tests, and summarize only the diff-relevant impact.
</PromptBlock>

<PromptBlock agent="reviewer">
Review the changes for regressions, edge-case gaps, and missing test coverage. Report findings by severity.
</PromptBlock>

## How To Use
1. Start with planning prompt and freeze scope.
2. Execute in small commits with validation after each checkpoint.
3. Run reviewer prompt before merge.
